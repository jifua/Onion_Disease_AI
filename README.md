# OnionBot AI Platform

AI-powered onion disease detection and smart agriculture chatbot, built with FastAPI, Deep Learning (ONNX Runtime), and a mini-RAG chatbot connected to Wikipedia and internal knowledge base.

## Features
- Onion leaf disease detection from image upload
- AI chatbot for agriculture questions (fusarium, alternaria, pests, cultivation)
- Prediction history and dashboard analytics
